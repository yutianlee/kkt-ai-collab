## Claims that need proof

### A1's claims requiring verification

1. **Integer-comparison part of the $n=1$ proof.** The final step reduces $E(6/5)<0.39$ to two exact integer comparisons after raising to the fifth power. The pair of large integers displayed has not been independently verified in this review, and the claim that the inequality simplifies to those numbers depends on the specific rational enclosure of $\pi$ and $e$ that A1 employed. These enclosures must be checked (e.g., $\pi>333/106$, $e^6<807/2$, $e^{344/985}<71/50$) and the power-raising algebra audited. Until that is done, the $n=1$ certificate should be treated as "near-certified" rather than permanently archived.

2. **Trigamma bound justification.** A1 uses $\psi'(x)<1/x+1/x^2$ without proof. While the bound is true and can be derived from an integral test, the lemma-bank version should supply a short proof (e.g., $\psi'(x)=1/x^2+\sum_{k=1}^\infty 1/(x+k)^2 < 1/x^2+\int_0^\infty dt/(x+t)^2 = 1/x^2+1/x$). This is a minor gap.

3. **Real-Stirling lower bound for $\Gamma(16/5)$.** The displayed inequality $\Gamma(x)>\sqrt{2\pi}\,x^{x-1/2}\exp(-x+1/(12x+1))$ is a known Robbins-type inequality (valid for $x>0$). Applying it at $x=16/5$ yields a numeric bound that must be shown to imply $\Gamma(16/5)>121/50$. The chain of rational comparisons used by A1 (enclosures of $\pi$, $e$, etc.) is plausible but has not been cross-checked in this review. This should be verified either by interval arithmetic or by an explicit rational expansion.

### A2's claims requiring proof

4. **Phase-shift bound.** A2 derives an explicit inequality $T_1^*-T_0 \le \frac{M_Q T_0^5}{16\Lambda_B^2(T_0^2-\alpha^2)}$ and then uses asymptotic estimates to claim the shift is $O(n^{-1/6})$ for $\alpha\le1.56\sqrt{n}$. The derivation relies on:
   - a Taylor expansion of $J_\alpha'$ with a Lagrange remainder that is not evaluated;
   - an estimate of $h'(z_1^*)/h(z_1^*)$ that uses the geometric Volterra bound at $z_0$ rather than at $z_1^*$, and treats $W_1(z)^2$ as decreasing on $[z_0,z_1^*]$ without justification.
   These steps are not proved. The claim that the lemma is `[PROVED]` is therefore an overstatement.

5. **Zero-safety argument.** Even if the phase-shift bound were rigorous, the argument that $T_1^*<j_{\alpha,1}$ depends on the asymptotic estimate $j_{\alpha,1}-j'_{\alpha,1}\approx1.05\alpha^{1/3}$ and on the specific numerical comparison $0.43 n^{-1/6} < 1.78$ for $n=10$. This is a floating-point heuristic, not a theorem. No explicit inequality is derived that works uniformly over the claimed regime.

6. **Tail variation $\varepsilon_{\mathrm{tail}}$.** The bound $\int_{z_0}^{z_1^*}W_1^2\,dx \le W_1(z_0)^2(z_1^*-z_0)$ assumes $W_1^2$ is non-increasing after the first peak, which is not proved. Moreover, the subsequent cancellation into a small $n^{-2/3}$ term is based on the same unverified approximations. Thus the "final scalar inequality" is not yet a theorem.

7. **Airy relative-amplitude sign.** A2 evaluates $K_{\tau\tau}$ and $K_{\tau\tau\tau}$ by approximating $p_B(u_0)\approx u_0$, $K_B'(u_0)\approx\Lambda_B$, etc., and then claims $\Psi_B(0)>0$. The actual removable value formula involves exact algebraic expressions; a rigorous sign analysis requires bounding the numerator $10\gamma K_{\tau\tau\tau}-9K_{\tau\tau}^2$ without dropping higher-order terms. The conclusion that $\Psi_B(0)>0$ may be correct, but it is not proved by the sketched approximation.

8. **Uniform bound $M_Q\le2.25$.** As noted, $\Lambda_B/B\le1$ is not proved in the full parameter range. A quick fix is to use $\Lambda_B\le B$? That would require a proof. The bound can be weakened without harm, but the present derivation is incomplete.

## Possible errors or hidden assumptions

### A1

- **Numerical rounding in endpoint comparisons.** The integer comparisons rely on exact rational numbers that are large but finite; there is no risk of floating-point error if the algebra is correctly executed. However, any slip in the exponentiation (raising both sides to the fifth power) could invalidate the conclusion. A symbolic algebra system audit is advisable.

- **Convexity argument domain.** The convexity proof holds for $\alpha\in[1/2,6/5]$, which is the range needed for the right-endpoint residual case $\alpha>1/2$. However, the theorem as stated covers $\frac12\le\alpha\le\frac65$; the upper bound $\alpha_E(1)=5/3>6/5$, so the range $6/5<\alpha\le5/3$ is not covered by this scalar proof. Moreover, for $\alpha>6/5$, $u_*=\alpha(\alpha+1)/(\alpha+2)$ exceeds $1$, so the scalar maximum may occur at the boundary $u=1$ instead. The proof does not address this; it assumes $u_*\le1$ holds for the whole range, but only verified for $\alpha\le6/5$. The $n=1$ theorem must either include the case $\alpha>6/5$ separately or rely on the central contour or energy module to clear it. The judge state from Round 25 already notes that the $n=1$ residual theorem covers $1/2\le\alpha\le6/5$; the rest is covered by other modules. So this is not an error but a scope note.

- **Gamma-ratio bound.** The reduction $H_1(u)^2\le\frac{u^\alpha(\alpha+1-u)^2}{\Gamma(\alpha+2)}$ uses $\frac{\Gamma(B)}{\Gamma(\beta+2)}\le B^\alpha$. The proof of that inequality uses $\psi(t)<\log t$, which is true for $t>0$. This yields $\frac{\Gamma(x+\alpha)}{\Gamma(x)} < (x+\alpha)^\alpha$ for $x=\beta+2$. Since $B=x+\alpha$, the bound becomes $\frac{\Gamma(B)}{\Gamma(\beta+2)}< B^\alpha$. The result is strict, not an inequality with equality. So it's correct.

- **Conditional rational-Bessel theorem.** The statement assumes $W_1>0$ on the integration interval, i.e., $2\sqrt{\Lambda_Bz}<j_{\alpha,1}$. No concrete criterion is given to ensure that the true critical point lies below that bound. This is correctly listed as an open hypothesis.

### A2

- **Over-labeling of lemmas.** Several claims are marked `[PROVED]` but are supported only by non-rigorous approximations (e.g., the phase-shift bound, the tail variation). This is a serious methodological issue; the status labels are misleading and could propagate unfounded conclusions.

- **Confusing local vs. global behavior.** The Volterra bound using monotonicity is correctly restricted to the pre-peak interval. However, the subsequent discussion of the phase shift implicitly uses the same geometric bound beyond the peak, which is not justified. The switching between rigorous and heuristic estimates is not clearly separated.

- **Inconsistent use of asymptotic notation.** Terms like $0.087\frac{\alpha^{11/3}}{\Lambda_B^2}$ and $0.43 n^{-1/6}$ appear without explicit error bounds. The numeric constants are not derived from fully controlled inequalities.

- **Gamma envelope dependency.** A2 acknowledges that the rational-Bessel theorem depends on a one-sided gamma envelope $M_{n,\alpha,B}\le 1+\varepsilon_\Gamma$, but no bound for $\varepsilon_\Gamma$ is supplied. The final scalar inequality $0.579<0.625$ is computed using the approximate constant $M_Q=2.25$ and $B_*^4\approx0.213$, but without rigorous control of the accumulated error (e.g., the tail factor $\exp(4\varepsilon_{\mathrm{tail}})$). This computation is a feasibility check, not a proof.

- **Airy relative-amplitude identity.** The derivation of $(a^2y_\zeta)_\zeta = \Psi_B a^2 y$ is algebraically correct if $a$ satisfies $a_{\zeta\zeta}+\zeta a=0$. However, the claim that $\Psi_B(0)>0$ is based on a fuzzy estimate: $K_{\tau\tau}(\tau_0)\approx\alpha^2/4$, $K_{\tau\tau\tau}(\tau_0)\approx\alpha^2/4$, which does not reflect the exact expressions. A proper sign analysis requires the actual formulas for $K_{\tau\tau}$ and $K_{\tau\tau\tau}$ derived in earlier rounds; these involve terms with $\Delta_B$ and are not simply $\approx\alpha^2/4$. Thus the conclusion is unsupported.
