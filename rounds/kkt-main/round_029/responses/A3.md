# Exact algebra checks
Every formula from the prompt and the Round 27 packet is re-verified.

**1. Derivation of \(p_B, q_B\).**
Start from the standard Jacobi self-adjoint form for the weighted function:
\[
\frac{d}{dx}\bigl((1-x^2)g'(x)\bigr) + \frac{F(x)}{1-x^2}g(x)=0,
\qquad F(x)=\kappa(1-x^2)-\frac{(d-sx)^2}{4},
\]
with \(s=\alpha+\beta\), \(d=\beta-\alpha\), \(\kappa=n(n+s+1)+s/2\).
Setting \(x=1-2u/B\) gives \(1-x^2 = \frac{4u}{B}(1-\frac{u}{B}) = \frac{4}{B}p_B(u)\) with \(p_B(u)=u(1-u/B)\).
Then
\[
\frac{d}{dx}\bigl((1-x^2)\frac{d}{dx}\bigr) = B\,\frac{d}{du}\bigl(p_B(u)\frac{d}{du}\bigr).
\]
Also \(d-sx = \beta-\alpha - s\bigl(1-\frac{2u}{B}\bigr) = -2\alpha + \frac{2su}{B}=2(r_B u-\alpha)\) because \(r_B = s/B\). Hence
\[
\frac{(d-sx)^2}{4} = (r_B u-\alpha)^2.
\]
Substituting into \(F\) and dividing by \(1-x^2\) yields
\[
\frac{F(x)}{1-x^2} = \kappa - \frac{(r_B u-\alpha)^2}{1-x^2}.
\]
Now \(\kappa/B = n + \frac{s}{2B} = n + \frac{r_B}{2} = c_B\). Therefore after dividing the ODE by \(B\),
\[
(p_B H')' + \Bigl(c_B - \frac{(r_B u-\alpha)^2}{4u(1-u/B)}\Bigr) H = 0.
\]
Thus \(q_B(u)\) is as stated. **Check passed.**

**2. Quadratic product \(K_B\).**
\[
K_B = p_B q_B = u(1-\tfrac{u}{B})
          \Bigl(c_B - \frac{(r_B u-\alpha)^2}{4u(1-u/B)}\Bigr)
        = c_B u(1-\tfrac{u}{B}) - \frac{(r_B u-\alpha)^2}{4}
\]
\[
= -\frac{\alpha^2}{4}
   + \bigl(c_B + \tfrac{\alpha r_B}{2}\bigr) u
   - \bigl(\tfrac{c_B}{B} + \tfrac{r_B^2}{4}\bigr) u^2.
\]
Hence \(\Lambda_B = c_B + \frac{\alpha r_B}{2}\), \(\Delta_B = \frac{c_B}{B} + \frac{r_B^2}{4}\). **Check passed.**

**3. Cap endpoint \(u_\sigma\).**
\[
\sigma = \frac{s}{s+2n},\qquad
u_\sigma = \frac{B(1-\sigma)}{2} = \frac{B}{2}\cdot\frac{2n}{s+2n}
        = \frac{nB}{s+2n} = \frac{nB}{B+n-1}.
\]
Using \(s = B-n-1\). For \(n=1\), \(u_\sigma = \frac{B}{B} = 1\); for \(n\ge2\) it is \(<n\). **Check passed.**

**4. Monotonicity of \(K_B\) on the cap.**
\(K_B\) is a concave quadratic; \(K_B'(u) = \Lambda_B - 2\Delta_B u\) is decreasing. Evaluating at \(u_\sigma\) after substituting the expressions yields (the algebra is lengthy but purely rational; it cancels to):
\[
K_B'(u_\sigma) = \frac{(\alpha+\beta)(n+\alpha+1)}{2B}
              = \frac{\alpha}{2} + \frac{\beta(n+1)}{2B}.
\]
Since \(\beta\ge0\), \(K_B'(u_\sigma) \ge \frac{\alpha}{2}\). In the residual strip \(\alpha>1/2\) we have \(K_B'(u) \ge \frac14\). This proves monotonicity on the cap. It does **not** by itself prove \(K_B(u_\sigma)>0\), since a strictly increasing function may remain negative throughout the interval. The endpoint sign must be checked separately when one wants a cap turning point. After clearing denominators,
\[
4(\alpha+\beta+2n)^2 K_B(u_\sigma)=N_{n,\alpha,\beta},
\]
where
\[
\begin{aligned}
N_{n,\alpha,\beta}={}&
4n^4+8n^3\alpha+8n^3\beta+3n^2\alpha^2+10n^2\alpha\beta+3n^2\beta^2\\
&-2n\alpha^3+2n\alpha\beta^2-\alpha^4-2\alpha^3\beta-\alpha^2\beta^2\\
&+4n^3+6n^2\alpha+6n^2\beta+2n\alpha^2+4n\alpha\beta+2n\beta^2 .
\end{aligned}
\]
Thus \(K_B(u_\sigma)>0\) is an additional algebraic inequality, not a consequence of monotonicity alone. In the residual strip used in the current proof state this endpoint positivity is expected and is easy to test exactly, but it should be recorded as a separate lemma or interval/rational check. **Derivative check passed; endpoint positivity requires the displayed sign check.**

**5. Forbidden-zone / no-turning-point clause (corrected).**
The phrase "no-turning-point" must be used relative to \(K_B\), not relative to the cap endpoint \(u_\sigma\). Since \(K_B(0)=-\alpha^2/4<0\) and \(K_B\) is increasing on the cap in the residual strip, the cases are:
- If \(K_B(u_\sigma)<0\), there is no zero of \(K_B\) in the cap.
- If \(K_B(u_\sigma)=0\), the first zero is exactly the interface point \(u_0=u_\sigma\).
- If \(K_B(u_\sigma)>0\), there is exactly one zero \(u_0\in(0,u_\sigma)\).
- Independently, \(K_B'(u)>0\) on \([0,u_\sigma]\), so \(K_B\) itself has no internal critical point in the cap.
- The first critical point \(u_1\) of the solution \(H\), if it exists in the cap, is a different object. It should not be identified with \(u_\sigma\) or with the zero of \(K_B\). In the first-lobe proof route one needs either a proof that such a \(u_1\) exists after \(u_0\), or a no-critical-point clause that moves the maximum to the cap boundary.

# Langer CAS cancellation audit
We outline the symbolic verification of the Liouville-Langer transformation near the turning point \(u=u_0\). The local independent variable is \(\tau = u-u_0\). Write \(K_B(u)=\gamma\tau + \frac{k_2}{2}\tau^2 + \frac{k_3}{6}\tau^3 + O(\tau^4)\) with
\[
\gamma = K_B'(u_0) > 0,\quad
k_2 = K_B''(u_0),\quad
k_3 = K_B'''(u_0)\,(\text{which is zero because }K_B\text{ is quadratic}).
\]
In fact \(K_B''(u) = -2\Delta_B\), constant, so \(k_2 = -2\Delta_B\) and all higher derivatives vanish. Consequently \(k_3=k_4=\dots=0\) and the expansion is exact up to \(O(\tau^4)\):

\[
K_B(u) = \gamma\tau - \Delta_B\,\tau^2.
\]

The Langer coordinate \(\zeta\) is defined by the differential equation \(\zeta \zeta_\tau^2 = K_B\).  We seek a series \(\zeta = c_1\tau + c_2\tau^2 + c_3\tau^3 + O(\tau^4)\) and solve the identity
\[
(c_1\tau + c_2\tau^2 + c_3\tau^3 + \cdots)(c_1 + 2c_2\tau + 3c_3\tau^2 + \cdots)^2 = \gamma\tau - \Delta_B\tau^2.
\]
Expanding and matching powers gives (exact rational coefficients performable by any CAS):

\[
\begin{aligned}
c_1 &= \gamma^{1/3},\\
c_2 &= \frac{k_2}{10\gamma^{2/3}} = -\frac{\Delta_B}{5\gamma^{2/3}},\\
c_3 &= \frac{25\gamma k_3 - 12k_2^2}{1050\gamma^{5/3}} = -\frac{12k_2^2}{1050\gamma^{5/3}} \quad (\text{since }k_3=0).
\end{aligned}
\]

The prompt's expression for \(c_2\) and \(c_3\) use \(k_2\) and \(k_3\); with our quadratic \(K_B\) we have \(k_3=0\) and therefore a simpler form. Substituting \(k_2 = -2\Delta_B\) yields
\[
c_2 = -\frac{\Delta_B}{5\gamma^{2/3}},\qquad
c_3 = -\frac{12\cdot4\Delta_B^2}{1050\gamma^{5/3}} = -\frac{48\Delta_B^2}{1050\gamma^{5/3}} = -\frac{8\Delta_B^2}{175\gamma^{5/3}}.
\]

Now the transformed ODE for \(\Psi(\zeta)\) (the scaled solution after the Langer transformation) contains a potential \(\Psi_B(\zeta)\) that arises from the Liouville substitution.  The singular terms at \(\zeta=0\) are \(\zeta^{-2}\) and \(\zeta^{-1}\) multiplied by quantities involving the derivative of the mapping. 1. Define the Taylor series of \(K\) and compute \(\zeta\) to order \(\tau^4\).
2. Invert the series to obtain \(\tau\) as a series in \(\zeta\): \(\tau = d_1\zeta + d_2\zeta^2 + d_3\zeta^3 + \cdots\).

# n=2 compactification audit
We derive the compactified cubic polynomial from first principles. The goal is to obtain a polynomial equation for the critical points of \(H\) for \(n=2\), expressed in terms of \(u\) (or \(t=u/B\)), \(\alpha\), and the compactified parameter \(\theta = (\alpha+3)/B\).  The Laguerre face \(\theta\rightarrow0\) is handled separately.

**Step 1: Explicit Jacobi polynomials.**
For \(n=2\), using the hypergeometric representation,
\[
P_2^{(\alpha,\beta)}(x) = \frac{(\alpha+1)_2}{2}
   \sum_{k=0}^2 \frac{(-2)_k (B)_k}{(\alpha+1)_k\,k!}\Bigl(\frac{1-x}{2}\Bigr)^k .
\]
With \(x=1-2u/B\) we have \(\frac{1-x}{2}=u/B\).  Then
\[
P_2(u) = \frac{(\alpha+1)(\alpha+2)}{2}
          - (\alpha+2)u + \frac{B+1}{2B} u^2.
\]
(Here and below we write \(P_2(u)\) for \(P_2^{(\alpha,\beta)}(1-2u/B)\).)

For the derivative equation we also need
\[
P_1^{(\alpha+1,\beta+1)}(u) = (\alpha+2) - \frac{B+1}{B} u .

Lemma L27.7 gives
\[
2(r_B u - \alpha)P_2(u) + 4u\Bigl(1-\frac{u}{B}\Bigr) P_1(u) = 0.
\]
Substituting the polynomials yields a cubic in \(u\).  Let us compute term by term.

2. **Wronskian conservation.**  If both \(Y\) and \(W_1\) satisfy linear ODEs of the form \(y'' + Q y = 0\) with different potentials \(Q_B\) and \(Q_{\text{ref}}\), then the derivative of the quotient \(h' = (W_1 Y' - Y W_1')/W_1^2\) involves the Wronskian \(W(Y,W_1)\).  The cancellation argument likely uses \(W = \text{constant}\) to convert a difference of products into an integral of the potential difference.
3. **Small initial deviation.**  At the turning point (\(\zeta=0\)) the solutions must be matched: \(Y(0)=W_1(0)\) and \(Y'(0)=W_1'(0)\) (or a scaled version).  That requires a precise Frobenius match, which in turn depends on the Langer cancellation guaranteeing that the error potential is integrable.
4. **Perturbation bound.**  The potential difference \(Q_B - Q_{\text{ref}}\) must be bounded by a known constant times something like \(1/(n+1)\).  The cancellation of \(\zeta^{-2},\zeta^{-1}\) ensures that the difference starts with finite \(\Psi_B(0)\) and is then smooth.
5. **No internal resonance.**  The first lobe must be sufficiently short that the Bessel reference remains dominant; a Gronwall inequality is then applied.  The maximum error in tracking the derivative is directly tied to the size of \(\Psi_B(0)\) and the interval length.

The highest risk in this route is the **global bound on the Wronskian integral** over the first lobe.  If the potential difference is not a sufficiently small multiple of the reference frequency, the derivative mismatch may accumulate.  Therefore a rigorous perturbation theorem (e.g., Olver's error bounds for second-order ODEs with a simple turning point) must be invoked with explicit constants.  That theorem is **not yet instantiated** for the KKT residual; its hypotheses (the analyticity strip, the size of the error potential, etc.) must be verified.  I recommend A2 produce a self-contained statement of the needed Olver/Dunster-Gil-Segura theorem and then audit its hypotheses against the computed \(\Psi_B(0)\) and the higher-order terms.

# Gamma envelope lemma audit
We need a lemma that gives a universal upper bound for the Bessel normalisation factor \(M_{n,\alpha,B}\) in the residual strip.  A concrete candidate statement is:

> **Lemma PL1' (Gamma-ratio envelope).** There exist absolute constants \(C_\Gamma>0\) and \(n_0\ge1\) such that for all integers \(n\ge n_0\) and all parameters satisfying \(\frac12\le\alpha\le\alpha_E(n)\) and \(\beta\ge0\),
> \[
> M_{n,\alpha,B} \le 1 + \frac{C_\Gamma}{n+1}.
> \]

*Proof strategy.*  Write
\[
M = \frac{\Gamma(n+\alpha+1)^{1/2}}{\Gamma(n+1)^{1/2}}\;
      \frac{\Gamma(B)^{1/2}}{\Gamma(B-\alpha)^{1/2}}\;
      (B\Lambda_B)^{-\alpha/2}.
\]
Use the standard bounds (Gautschi/Kershaw):
\[
x^{1-s} \le \frac{\Gamma(x+1)}{\Gamma(x+s)} \le (x+1)^{1-s} \quad (0<s<1),
\]
together with the duplication formula when \(\alpha\) is large.  Split the \((\alpha,\beta)\) plane into regimes:
- \(\alpha\le A\) (bounded \(\alpha\)): use simple factorial bounds; \(M\) stays near 1.
- \(A\le\alpha\le c n\): use Stirling's formula with error term (Binet type) to estimate the ratios.
- \(\alpha\) close to \(\alpha_E(n)\) (i.e. \(\alpha\sim C n\)): asymptotically \(M\) may approach a constant less than 1.

The worst-case excess occurs when \(\alpha\) is small and \(\beta\) is zero, where numerics suggest \(M\lesssim 1+0.002/(n+1)\); a uniform bound like \(1+2/(n+1)\) seems very plausible.  However, the exact constants \(n_0\) and \(C_\Gamma\) have not been derived.  The standard Wendel/Gautschi inequalities alone are too crude to capture the \(O(1/n)\) behavior when \(\alpha\) grows linearly with \(n\); a finer asymptotic expansion (e.g., Erdelyi's asymptotic for the ratio of two Gamma functions) is required.

**Verdict:** The lemma is mathematically sound in principle, but the regime-splitting and the choice of constants remain **unexecuted**.  A quantitative interval-arithmetic scan (see Reproducible verification tasks) could provide numerical evidence and guide the proof.

# Theorem-dependency checks
The proof sketch depends on the following external theorems:
- **Jacobi differential equation and its self-adjoint form.**  Accepted as known.
- **Haagerup-Schlichtkrull central-contour clearance.**  This is used to reduce the residual problem to the cap; stated in the KKT literature.
- **Small-exponent endpoint theorem (\(\alpha\le1/2\)).**  This is a separate piece and not audited here.
- **Landau's monotonicity of the Bessel maximum.**  The original Landau statement is "the absolute maximum of \(J_\nu(x)\) is a strictly decreasing function of \(\nu\) for \(\nu\ge0\)".  The exact formulation and its proof must be quoted; the consequence for \(\nu\ge1/2\) gives a bound \(<0.680\) once the \(\nu=1/2\) case is checked.
- **Wendel, Gautschi, Kershaw inequalities.**  These provide rigorous bounds for Gamma ratios.  Their range of validity must be matched to the parameter regimes.
- **Olver's Theorem 11.1 (or Dunster-Gil-Segura).**  This is the main perturbation theorem for ODEs with a simple turning point.  It provides a bound of the form
  \[
  \bigl| Y(\zeta) - \mathcal{B}(\zeta) \bigr| \le \frac{C}{|\zeta|^{1/4}} \int_0^\zeta \frac{|\Psi_B(t)|}{(1+|t|)\,t^{3/4}} dt \;\times\; \text{(weight function)}.
  \]
  The exact statement and constants must be instantiated for the KKT equation.
- **Gronwall/Bellman inequality.**  Needed to propagate error across the first lobe.

Every one of these theorems must be quoted with its precise hypotheses before the corresponding step in the proof can be considered closed.  The missing step is the **instantiation of the Olver theorem**: the error potential \(\Psi_B\) must be bounded in absolute value by a constant multiple of \(1/(n+1)\) on the relevant interval, and the weight functions must be shown to be integrable.  That is the single largest dependency.

# Hidden assumptions and failure modes
1. **Positive derivative of \(K_B\) on the cap.**  Used to guarantee a single turning point and to bound \(K_B'(u)\) from below.  This holds strictly under the stated assumptions; no hidden singularity.
2. **Regular solution at \(u=0\).**  We assume \(\alpha>0\) so that the exponent \(\alpha/2>0\); the special case \(\alpha=0\) is excluded from the residual strip.  The Frobenius expansion is standard.
3. **No double turning point.**  Because \(K_B\) is quadratic and \(K_B'(u)>0\) on the cap, the only zero is simple; there is no degenerate case \(K_B(u_0)=K_B'(u_0)=0\).
4. **Liouville transformation singularity at \(u=0\).**  The transformation \(Y = p_B^{1/2} H\) becomes singular at \(u=0\) because \(p_B\sim u\).  This is harmless because the solution behaves like \(u^{\alpha/2}\) and the exponent \(\alpha/2\) leads to a well-behaved \(Y\).
5. **Langer expansion convergence.**  The series for \(\zeta\) is asymptotic; for matching at the turning point we only need a few terms.  The cancellation of the leading singular terms is algebraic and does not rely on convergence; however, the error in truncating higher orders must be controlled.  A full Olver-type theorem handles this automatically.
6. **Potential failure of the Bessel-envelope when \(M>1\).**  The normalisation factor \(M\) can exceed 1 by a small amount (observed numerically).  If the excess is larger than the slack allowed by the perturbation bound, the first-lobe maximum could surpass the desired threshold.  This is why the gamma-envelope lemma must be sharp.
7. **Radial sparsity of the residual strip.**  For very large \(n\), the upper boundary \(\alpha_E(n)\sim n\) while \(\beta\) is allowed to be arbitrarily large; the compactified \(\theta = (\alpha+3)/B\) can approach 0 or 1, potentially affecting the size of the perturbation.  The analysis must handle the full product \((n,\alpha,\beta)\) space.

# Divergent alternatives and 20% exploration
I allocate about 20% of this report to a **Sonin-Krasikov-style envelope** applied to the \(n=2\) box.  The idea is to construct a quadratic form \(V(u)=A(u)H^2 + B(u)(p_B H')^2 + C(u) H\,p_B H'\) whose derivative along the ODE has a definite sign.  If \(V'\le0\) and \(V(0)\) is known, one obtains a pointwise bound on \(H\).

**Attempt for \(n=2\).**  Let \(p=P_B\), \(q=q_B\).  For any smooth functions \(A,B,C\) we compute
\[
V' = \bigl(A' - C q\bigr) H^2
    + \bigl(A + p B' + p' B - C p\bigr) 2 H H'
    + \bigl(p C' + p' C + 2 B q - p B'\bigr) (H')^2.
\]
(using \((p H')' = -q H\)).  We would like to choose \(A,B,C\) so that the coefficient of \(H H'\) vanishes and the \((H')^2\) coefficient is non-positive, while the \(H^2\) coefficient has a known bound.

A canonical choice is \(B = \frac{A}{p}\), \(C=0\); then \(V = A(H^2 + (p H')^2/p)\).  But this gives no control.  A more flexible ansatz is
\[
A(u) = a_0 + a_1 u,\quad
B(u) = \frac{b_0 + b_1 u}{p(u)},\quad
C(u) = c_0 + c_1 u.
\]
For \(n=2\), the ODE coefficients are rational; one could attempt to solve the system of algebraic equations so that the derivative is non-positive.  This is a small-scale symbolic problem that can be tackled by a CAS.  I have not executed it.

**Possible outcome:** For certain parameter ranges one might find a Lyapunov function that certifies the bound without any Bessel analysis.  If successful for \(n=2\), the same ansatz could be generalized to higher \(n\) by an SOS (sum-of-squares) relaxation on the polynomial coefficients of the ODE.  This would be a genuinely divergent route, avoiding the entire Langer perturbation machinery.

Because the exploration time is limited, I conclude with a **clear failure reason**: I was unable to produce a working Sonin envelope by hand; the needed positivity conditions involve the sign of a complicated rational function on the cap, and without a computer algebra session I cannot verify it.  The pilot is therefore marked **unexecuted** but remains a worthy small project.
