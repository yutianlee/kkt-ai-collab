import sympy as sp

def verify_langer_residual_limit():
    t = sp.Symbol('t', real=True)
    gamma, a, b = sp.symbols('gamma a b', real=True, positive=True)

    # K(tau) expansion near tau_0 (where t = tau - tau_0)
    K = gamma*t + a*pow(t, 2) + b*pow(t, 3)

    # We need zeta(t) such that (2/3)zeta^(3/2) = integral(sqrt(K))
    # sqrt(K) = sqrt(gamma*t) * sqrt(1 + (a/gamma)*t + (b/gamma)*t^2)
    sqrt_K = sp.sqrt(gamma*t) * (1 + (a/(2*gamma))*t + (b/(2*gamma) - pow(a, 2)/(8*pow(gamma, 2)))*pow(t, 2))

    # Integrate to get (2/3)zeta^(3/2)
    integral_sqrt_K = sp.sqrt(gamma) * ((2/3)*pow(t, sp.Rational(3, 2)) + (a/(5*gamma))*pow(t, sp.Rational(5, 2)) + (b/(7*gamma) - pow(a, 2)/(28*pow(gamma, 2)))*pow(t, sp.Rational(7, 2)))

    # Solve for zeta
    zeta_3_2 = (3/2) * integral_sqrt_K
    zeta = pow(zeta_3_2, sp.Rational(2, 3))

    # Compute derivatives with respect to t (which is d/dtau)
    K_t = sp.diff(K, t)
    K_tt = sp.diff(K_t, t)

    # Psi_B(zeta) = (zeta * K_tt) / (4 * K^2) - (5 * zeta * K_t^2) / (16 * K^3) + 5 / (16 * zeta^2)
    term1 = (zeta * K_tt) / (4 * pow(K, 2))
    term2 = (5 * zeta * pow(K_t, 2)) / (16 * pow(K, 3))
    term3 = 5 / (16 * pow(zeta, 2))

    Psi_B = term1 - term2 + term3

    # Series expansion around t=0
    Psi_series = sp.series(Psi_B, t, 0, 1)

    print("Limit of Psi_B as t -> 0:")
    sp.pprint(Psi_series.removeO().simplify())

verify_langer_residual_limit()

In the Robbins-based gamma-ratio envelope, A4 assumes that the trapezoidal error term $\frac{\alpha^3}{12} \max_{t \in [B-\alpha, B]} |f''(t)|$ is sufficiently small to fit the KKT margin, but does not explicitly evaluate it. The Robbins formulation is the correct path for finite-parameter gamma control. However, A4 must finish the derivation.

Let $f(t) = (t-0.5)\ln t - t$. The first derivative is $f'(t) = \ln t - 0.5/t$. The second derivative is $f''(t) = \frac{1}{t} + \frac{1}{2t^2}$.
Since $f''(t)$ is strictly decreasing for $t > 0$, the maximum on the interval $[B-\alpha, B]$ occurs at the lower bound, $t = B-\alpha$.
Thus, the trapezoidal error is strictly bounded by:
$$ \mathcal{E} \le \frac{\alpha^3}{12} \left( \frac{1}{B-\alpha} + \frac{1}{2(B-\alpha)^2} \right) $$

This is a massive mathematical point that dictates the regime split. If we are in the bulk transition strip where $\alpha = c n$ and $B = \mathcal{O}(n)$, then $B-\alpha = \mathcal{O}(n)$. The error term scales as:
$$ \mathcal{E} \approx \frac{(cn)^3}{12} \left( \frac{1}{\mathcal{O}(n)} \right) = \mathcal{O}(n^2) $$
An error of $\mathcal{O}(n^2)$ completely destroys the gamma-ratio bound. It does not go to zero; it diverges quadratically.

Conversely, if we are in the rational-Bessel regime where $\alpha \le C\sqrt{n}$, the error scales as:
$$ \mathcal{E} \approx \frac{(C\sqrt{n})^3}{12} \left( \frac{1}{\mathcal{O}(n)} \right) = \mathcal{O}(n^{-1/2}) $$
This error goes to zero as $n \to \infty$.

Therefore, A4's Robbins formulation gives strong evidence that the regime split is mandatory. The Robbins bound appears suitable for the small-$\alpha$ track after constants are fixed, but its displayed error term grows too quickly for the bulk Langer track. A4 needs to write this out explicitly.
