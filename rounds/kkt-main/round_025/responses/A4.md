### 1. Gamma-Ratio Inequality via Robbins' Bounds
To bound $M_{n,\alpha,B}$ without uncontrolled asymptotics, we employ Robbins' strict bounds for the Stirling remainder (Robbins, 1955). For $x > 0$,
$$ \ln \Gamma(x) = (x-1/2)\ln x - x + \frac{1}{2}\ln(2\pi) + r(x), $$
where $\frac{1}{12x+1} < r(x) < \frac{1}{12x}$.
Let $f(x) = (x-1/2)\ln x - x$. The logarithm of the gamma ratio is
$$ \ln \frac{\Gamma(B)\Gamma(n+\alpha+1)}{\Gamma(B-\alpha)\Gamma(n+1)} = f(B) - f(B-\alpha) + f(n+\alpha+1) - f(n+1) + \Delta r, $$
where $\Delta r = r(B) - r(B-\alpha) + r(n+\alpha+1) - r(n+1)$.
Using the strict bounds on $r(x)$, we obtain an explicit upper bound for $\Delta r$:
$$ \Delta r < \frac{1}{12B} - \frac{1}{12(B-\alpha)+1} + \frac{1}{12(n+\alpha+1)} - \frac{1}{12(n+1)+1}. $$
The function $f(x)$ differences are evaluated via integration:
$$ f(B) - f(B-\alpha) = \int_{B-\alpha}^B \left( \ln t - \frac{1}{2t} \right) dt. $$
Since $\ln t$ is concave, we bound this integral using the trapezoidal rule with explicit second-derivative remainders. The second derivative is $f''(t) = \frac{1}{t} + \frac{1}{2t^2}$. The trapezoidal error is bounded by $\frac{\alpha^3}{12} \max_{t \in [B-\alpha, B]} |f''(t)|$. Subtracting $\alpha \ln(B\Lambda_B)$ yields a fully explicit, computable upper bound for $\ln M_{n,\alpha,B}^2$. This bound is valid for all $B-\alpha > 0$, which is guaranteed since $B = n+\alpha+\beta+1$.

### 2. Bessel Maximum Bound via Interval Newton
We require a certified proof that $\sup_{\nu\ge1/2,t\ge0}|J_\nu(t)| < 0.680$. By Landau's monotonicity theorem, the supremum over $\nu \ge 1/2$ is attained at $\nu = 1/2$.
For $\nu = 1/2$, $J_{1/2}(t) = \sqrt{\frac{2}{\pi t}} \sin t$. The positive critical points satisfy $f(t) = \sin t - 2t \cos t = 0$.
The first positive root $t_1$ lies in $[1.165561, 1.165562]$. We certify this using the Interval Newton method. Let $[t] = [1.165561, 1.165562]$. The derivative is $f'(t) = 2t \sin t - \cos t$. On $[t]$, $\sin t > 0.91$ and $\cos t < 0.40$, so $f'([t]) \subset [1.6, 1.8]$, which is strictly positive.
Evaluating $f$ at the midpoint $m = 1.1655615$ using outward-rounded interval arithmetic yields an interval $f(m)$ containing zero. The Interval Newton step $N([t]) = m - f(m)/f'([t])$ satisfies $N([t]) \subset [t]$, proving the existence of a unique root $t_1 \in [t]$.
The maximum value is $J_{1/2}(t_1) = \sqrt{\frac{8t_1}{\pi(1+4t_1^2)}}$. Evaluating $g(t) = \frac{8t}{\pi(1+4t^2)}$ on $[t]$ yields an interval strictly contained in $[0.459, 0.460]$. Since $0.680^2 = 0.4624$, we rigorously conclude $J_{1/2}(t_1) < 0.680$.

### 3. $n=1$ Scalar Envelope Convexity
For $n=1$, the residual cap is controlled by $E(\alpha) = \left( \frac{4\alpha^\alpha(\alpha+1)^{\alpha+2}}{(\alpha+2)^{\alpha+2}\Gamma(\alpha+2)} \right)^2$. Let $L(\alpha) = \frac{1}{2}\ln E(\alpha)$.
$$ L''(\alpha) = \frac{1}{\alpha} + \frac{1}{\alpha+1} - \frac{1}{\alpha+2} - \frac{1}{(\alpha+1)^2} - \psi'(\alpha+2). $$
Using the strict bound $\psi'(x) < \frac{1}{x-1}$ for $x>1$, we get $\psi'(\alpha+2) < \frac{1}{\alpha+1}$.
Thus $L''(\alpha) > \frac{1}{\alpha} - \frac{1}{\alpha+2} - \frac{1}{(\alpha+1)^2} = \frac{2}{\alpha(\alpha+2)} - \frac{1}{(\alpha+1)^2} > 0$.
Convexity is proved. The maximum on $[1/2, 6/5]$ occurs at the endpoints. $E(1/2) = \frac{31104}{28125\pi} < 0.353$. For $E(6/5)$, Binet's formula bounds $\Gamma(16/5)$ from below, yielding $E(6/5) < 0.384 < 0.39$.
