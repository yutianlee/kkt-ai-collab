### Rational-Coordinate Transformation and Normal Form

We rigorously transform the affine coordinate ODE into the rational coordinate normal form to eliminate geometric amplitude inflation near the boundary.
Define the affine variable $u$ and the rational coordinate $z$:

$$z = \frac{Bu}{B-u} \implies u = \frac{Bz}{B+z}$$

We compute the necessary coordinate derivatives with respect to $z$:

$$B-u = B - \frac{Bz}{B+z} = \frac{B^2}{B+z}$$

$$\frac{du}{dz} = \frac{B(B+z) - Bz}{(B+z)^2} = \frac{B^2}{(B+z)^2}$$

The certified endpoint ODE is defined as:

$$\frac{d}{du}\left( p_B(u) \frac{dH}{du} \right) + q_B(u)H = 0$$

where the kinetic coefficient is $p_B(u) = u(1 - u/B)$. Let us express $p_B(u)$ entirely in terms of $z$:

$$p_B(u(z)) = \frac{Bz}{B+z} \left( 1 - \frac{z}{B+z} \right) = \frac{Bz}{B+z} \left( \frac{B}{B+z} \right) = \frac{B^2 z}{(B+z)^2}$$

Now we convert the kinetic differential operator using the chain rule $\frac{d}{du} = \frac{dz}{du} \frac{d}{dz}$:

$$\frac{dz}{du} = \left( \frac{du}{dz} \right)^{-1} = \frac{(B+z)^2}{B^2}$$

$$p_B(u) \frac{dH}{du} = \left[ \frac{B^2 z}{(B+z)^2} \right] \left[ \frac{(B+z)^2}{B^2} \frac{dH}{dz} \right] = z \frac{dH}{dz}$$

Substitute this back into the outer derivative operator $\frac{d}{du}$:

$$\frac{d}{du}\left( p_B(u) \frac{dH}{du} \right) = \frac{dz}{du} \frac{d}{dz} \left( z \frac{dH}{dz} \right) = \frac{(B+z)^2}{B^2} \frac{d}{dz} \left( z \frac{dH}{dz} \right)$$

The full ODE in the $z$ coordinate becomes:

$$\frac{(B+z)^2}{B^2} \frac{d}{dz}\left( z \frac{dH}{dz} \right) + q_B(u(z))H = 0$$

Divide through by the leading rational factor to isolate the derivative:

$$\frac{d}{dz}\left( z \frac{dH}{dz} \right) + \frac{B^2}{(B+z)^2} q_B(u(z)) H = 0$$

Let us define the scaled potential $\widehat{q}_B(z) = \frac{B^2}{(B+z)^2} q_B(u(z))$. Observe the invariant identity:

$$z \widehat{q}_B(z) = z \frac{B^2}{(B+z)^2} q_B(u(z)) = p_B(u(z)) q_B(u(z)) = K_B(u(z))$$

To remove the first derivative, we introduce the Liouville normal form substitution $Y_z(z) = z^{1/2}H(u(z))$. Differentiating yields:

$$H' = z^{-1/2}Y_z' - \frac{1}{2}z^{-3/2}Y_z \implies zH' = z^{1/2}Y_z' - \frac{1}{2}z^{-1/2}Y_z$$

Differentiating $zH'$ with respect to $z$:

$$(zH')' = \frac{1}{2}z^{-1/2}Y_z' + z^{1/2}Y_z'' + \frac{1}{4}z^{-3/2}Y_z - \frac{1}{2}z^{-1/2}Y_z' = z^{1/2}Y_z'' + \frac{1}{4}z^{-3/2}Y_z$$

Substituting into the ODE $(zH')' + \widehat{q}_B(z)H = 0$:

$$z^{1/2}Y_z'' + \frac{1}{4}z^{-3/2}Y_z + \widehat{q}_B(z)z^{-1/2}Y_z = 0$$

Dividing out the $z^{1/2}$ factor, we secure the normal form:

$$Y_z'' + \left( \frac{1}{4z^2} + \frac{\widehat{q}_B(z)}{z} \right) Y_z = 0$$

Because $\widehat{q}_B(z)/z = z\widehat{q}_B(z)/z^2 = K_B(u(z))/z^2$, the potential takes the exact form:

$$Q_z(z) = \frac{K_B(u(z)) + 1/4}{z^2}$$

This line-by-line derivation is labeled `[PROVED]`. It definitively establishes the correct positive Liouville normal form sign, confirming the shift is rigorously $+1/4$.
