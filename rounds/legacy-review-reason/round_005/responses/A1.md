## 1. Evaluation of A2-4.md and A3-4.md

**A2-4.md is useful diagnostically but not a proof.** It correctly rejects the shifted Sonin denominator
\[
S_\lambda=g^2+\frac{A^2g'^2}{F+\lambda A^2}
\]
because the derivative has an unavoidable mixed term
\[
\frac{2\lambda A^2}{F+\lambda A^2}gg',
\]
and because \(F+\lambda A^2\) cannot be made positive at both endpoints when \(\alpha,\beta>0\). That critique is correct. A2-4 also correctly identifies the direct \(x\)-Sonin localization plus endpoint Bessel comparison as the right architecture. But its claimed final closure is not proved. The statement that the large negative \(\beta^2\)-term "perfectly annihilates" against \(N^2\) is only an asymptotic observation; it does not prove the required scalar inequality
\[
E_R\le T_{n,\alpha,\beta}.
\]
So A2-4 overclaims. It gives no verifiable inequality chain closing the endpoint constants.

**A3-4.md is more reliable.** It correctly says the endpoint criterion reduces the problem to explicit scalar inequalities and does not honestly close the intermediate region. Its main weakness is that it calls the last verification "routine"; it is not routine. The remaining endpoint inequalities are exactly where sharpness is lost. A3-4 is therefore useful as a roadmap, not as a completed proof.

The KKT review remains consistent with this: KKT prove the refined estimate for integer parameters and state the real-parameter version as Conjecture 6.1; the review also notes that the full real \(\alpha,\beta\ge0\) case appears open. The original KKT paper is the source for the conjecture and the integer-parameter result. ([Research Explorer][1])

# 2. What survives from the previous route

Let
\[
g(x)=g_n^{(\alpha,\beta)}(x)
=
C_{n,\alpha,\beta}
\left(\frac{1-x}{2}\right)^{\alpha/2}
\left(\frac{1+x}{2}\right)^{\beta/2}
P_n^{(\alpha,\beta)}(x),
\]
where
\[
C_{n,\alpha,\beta}^2
=
\frac{\Gamma(n+1)\Gamma(n+\alpha+\beta+1)}
{\Gamma(n+\alpha+1)\Gamma(n+\beta+1)}.
\]

Set
\[
A(x)=1-x^2,\qquad s=\alpha+\beta,\qquad d=\beta-\alpha,
\]
\[
N=n+\frac{s+1}{2},
\qquad
\kappa=n(n+s+1)+\frac{s}{2}.
\]

Then
\[
\boxed{
(Ag')'+\frac{F(x)}{A(x)}g=0,
}
\]
where
\[
\boxed{
F(x)
=
\kappa(1-x^2)
-\frac14\bigl(d-sx\bigr)^2.
}
\]

On the interval where \(F>0\), the correct Sonin functional is
\[
\boxed{
S_g(x)=g(x)^2+\frac{A(x)^2g'(x)^2}{F(x)}.
}
\]
It satisfies
\[
\boxed{
S_g'(x)
=
-\frac{A(x)^2F'(x)}{F(x)^2}g'(x)^2.
}
\]

Since \(F\) is a concave quadratic, \(S_g\) decreases from the left turning point to the vertex of \(F\), and increases from the vertex to the right turning point. Hence all interior extrema are dominated by the two outer endpoint lobes. This part is solid.

The regularized identity
\[
\boxed{
W(x):=F(x)g(x)^2+A(x)^2g'(x)^2,
\qquad
W'(x)=F'(x)g(x)^2
}
\]
is also solid. It is the correct way to cross the turning points; the shifted \(S_\lambda\) construction is not.

# 3. A new closed endpoint theorem

The main new progress I can make is this:
\[
\boxed{
\text{The endpoint lobe at }x=1\text{ is controlled for all }\beta\ge0
\text{ whenever }0\le\alpha\le\frac12.
}
\]

By symmetry, the endpoint lobe at \(x=-1\) is controlled for all \(\alpha\ge0\) whenever
\[
0\le\beta\le\frac12.
\]

This substantially extends the earlier small square
\[
0\le\alpha,\beta\le\frac12.
\]

---

## 3.1 Right-endpoint Liouville equation

Put
\[
x=\cos\theta,\qquad 0<\theta<\pi,
\]
and
\[
v(\theta)=\sqrt{\sin\theta}\,g(\cos\theta).
\]

Then
\[
\boxed{
v''(\theta)+Q(\theta)v(\theta)=0,
}
\]
with
\[
\boxed{
Q(\theta)
=
N^2
+
\frac{1/4-\alpha^2}{4\sin^2(\theta/2)}
+
\frac{1/4-\beta^2}{4\cos^2(\theta/2)}.
}
\]

Let
\[
\sigma=\frac{s}{s+2n},
\qquad
\Theta=\arccos\sigma.
\]
Then \(0<\Theta\le\pi/2\) unless \(s=0\), which is the Legendre case.

Define
\[
B(\theta)=\frac{1}{4\sin^2(\theta/2)}-\frac1{\theta^2},
\qquad
C(\theta)=\frac1{4\cos^2(\theta/2)}.
\]
On \(0<\theta\le\Theta\), both \(B\) and \(C\) are increasing.

For \(0\le\alpha\le1/2\), set
\[
M_R^2
=
N^2+\frac{1/4-\alpha^2}{12}+m_\beta,
\]
where
\[
m_\beta=
\begin{cases}
\dfrac{1/4-\beta^2}{4}, & 0\le\beta\le\frac12,\\[1.2ex]
\left(\dfrac14-\beta^2\right)C(\Theta), & \beta>\frac12.
\end{cases}
\]

Then
\[
\boxed{
Q(\theta)
\ge
M_R^2+\frac{1/4-\alpha^2}{\theta^2}
\qquad(0<\theta\le\Theta).
}
\]

The regular comparison solution is
\[
z(\theta)=K_R\sqrt{\theta}J_\alpha(M_R\theta),
\]
where endpoint matching gives
\[
\boxed{
K_R=
C_{n,\alpha,\beta}
\frac{\Gamma(n+\alpha+1)}
{\Gamma(n+1)M_R^\alpha}.
}
\]

Singular Sturm comparison gives, through the relevant endpoint part of the first lobe,
\[
\boxed{
0\le g(\cos\theta)
\le
K_R
\left(\frac{\theta}{\sin\theta}\right)^{1/2}
J_\alpha(M_R\theta).
}
\]

## 3.2 Bessel envelope

For \(0\le z\le j_{\alpha,1}\),
\[
J_\alpha(z)
\le
\frac{(z/2)^\alpha}{\Gamma(\alpha+1)}
\exp\!\left(-\frac{z^2}{4(\alpha+1)}\right).
\]

Also, since \(0<\Theta\le\pi/2\),
\[
\left(\frac{\theta}{\sin\theta}\right)^{1/2}
\le
\exp(\rho\theta^2),
\qquad
\rho\le\frac{3}{32}.
\]

Thus
\[
|g(\cos\theta)|
\le
D_R\theta^\alpha e^{-c_R\theta^2},
\]
where
\[
D_R=
\frac{
C_{n,\alpha,\beta}\Gamma(n+\alpha+1)
}
{
\Gamma(n+1)2^\alpha\Gamma(\alpha+1)
},
\]
and
\[
c_R=
\frac{M_R^2}{4(\alpha+1)}-\rho.
\]

The scalar maximum gives
\[
|g(\cos\theta)|
\le
D_R
\left(
\frac{\alpha}{2ec_R}
\right)^{\alpha/2},
\]
with the usual limiting interpretation when \(\alpha=0\).

---

## 3.3 The key lower bound for \(c_R\)

For \(0\le\alpha\le1/2\) and \(\beta\ge0\), one has
\[
\boxed{
c_R
\ge
\frac{(n+1)(n+\beta+1)}{4e}.
}
\]

A convenient sufficient algebraic inequality is
\[
8M_R^2
\ge
3(\alpha+1)\bigl((n+1)(n+\beta+1)+1\bigr),
\]
together with
\[
\rho\le\frac{3}{32},
\qquad
e>\frac83.
\]

For \(0\le\beta\le1/2\), the difference is positive because
\[
8M_R^2-3(\alpha+1)\bigl((n+1)(n+\beta+1)+1\bigr)
\]
is increasing in \(\beta\), decreasing in \(\alpha\), and at \(\beta=0,\alpha=1/2\) equals
\[
\frac72n^2+3n-4>0
\qquad(n\ge1).
\]

For \(\beta\ge1/2\), write the difference as
\[
\frac{P(n,\alpha,\beta)}{6(n+\alpha+\beta)}.
\]
The numerator \(P\) is convex and increasing in \(\beta\) for \(\beta\ge1/2\), and at \(\beta=1/2\) factors as
\[
P(n,\alpha,1/2)
=
\frac{2\alpha+2n+1}{2}
\left(
8\alpha^2-18\alpha n^2+3\alpha n-9\alpha
+30n^2+27n-17
\right).
\]
The second factor is minimized at \(\alpha=1/2\), where it becomes
\[
21n^2+\frac{57}{2}n-\frac{39}{2}>0.
\]
Hence the lower bound for \(c_R\) follows.

## 3.4 Gamma-ratio step

By Wendel/Gautschi-type gamma-ratio inequalities, for \(0\le\alpha\le1\),
\[
\frac{\Gamma(n+\alpha+1)}{\Gamma(n+1)}
\le
(n+1)^\alpha,
\]
and
\[
\frac{\Gamma(n+\alpha+\beta+1)}{\Gamma(n+\beta+1)}
\le
(n+\beta+1)^\alpha.
\]

Therefore
\[
D_R
\le
\frac{
\bigl((n+1)(n+\beta+1)\bigr)^{\alpha/2}
}
{
2^\alpha\Gamma(\alpha+1)
}.
\]

Using the lower bound for \(c_R\),
\[
D_R
\left(
\frac{\alpha}{2ec_R}
\right)^{\alpha/2}
\le
\frac1{\Gamma(\alpha+1)}
\left(\frac{\alpha}{2}\right)^{\alpha/2}.
\]

Finally, for \(0\le\alpha\le1/2\),
\[
\boxed{
\frac1{\Gamma(\alpha+1)}
\left(\frac{\alpha}{2}\right)^{\alpha/2}
\le
\left(\frac{2}{2+\alpha}\right)^{1/4}.
}
\]
One proof is to define
\[
\Phi(\alpha)
=
\log\Gamma(1+\alpha)
+\frac14\log\frac{2}{2+\alpha}
-\frac{\alpha}{2}\log\frac{\alpha}{2}.
\]
Then \(\Phi(0)=0\), and using
\[
\psi(1+\alpha)\ge \log\left(\alpha+\frac12\right)
\]
one obtains \(\Phi'(\alpha)\ge0\) on \([0,1/2]\).

Moreover,
\[
T_{n,\alpha,\beta}^4
=
1-\frac{\alpha\beta}{(n+\alpha+1)(n+\beta+1)}
\ge
1-\frac{\alpha}{n+\alpha+1}
=
\frac{n+1}{n+\alpha+1}
\ge
\frac{2}{2+\alpha}.
\]
Thus
\[
\boxed{
|g_n^{(\alpha,\beta)}(x)|
\le
T_{n,\alpha,\beta}
}
\]
on the right endpoint lobe whenever
\[
0\le\alpha\le\frac12,\qquad \beta\ge0.
\]

By symmetry,
\[
\boxed{
|g_n^{(\alpha,\beta)}(x)|
\le
T_{n,\alpha,\beta}
}
\]
on the left endpoint lobe whenever
\[
0\le\beta\le\frac12,\qquad \alpha\ge0.
\]

# 4. Consequences now proved

Combining direct Sonin localization, the central contour region, the weighted-energy estimate, and the new endpoint theorem gives a strictly larger proven region than before.

The refined KKT inequality is proved in the following cases:

### Case A: central oscillatory region

If
\[
[x_-,x_+]\subseteq[-\sigma,\sigma],
\qquad
\sigma=\frac{\alpha+\beta}{\alpha+\beta+2n},
\]
then the bound follows from the central branch-safe contour estimate plus Sonin localization.

### Case B: small endpoint exponent overflow

If the right endpoint overflows but
\[
0\le\alpha\le\frac12,
\]
the new endpoint theorem controls the right lobe.

If the left endpoint overflows but
\[
0\le\beta\le\frac12,
\]
the symmetric endpoint theorem controls the left lobe.

Thus, in particular, all cases with
\[
0\le\alpha,\beta\le\frac12
\]
remain fully proved, and the proof now also covers many asymmetric cases such as
\[
0\le\alpha\le\frac12,\quad \beta\gg1,
\]
provided the opposite endpoint is either central or handled by energy.

### Case C: weighted-energy region

For \(\alpha,\beta>0\), the exact weighted-energy estimate gives
\[
|g(x)|^4
\le
\frac{\alpha+\beta}{2\alpha\beta}
\left(
n+\frac12-\frac{1}{2(2n+\alpha+\beta+1)}
\right).
\]
Therefore the refined KKT inequality follows whenever
\[
\boxed{
\frac{\alpha+\beta}{2\alpha\beta}
\left(
n+\frac12-\frac{1}{2(2n+\alpha+\beta+1)}
\right)
\le
\frac{(n+1)(n+\alpha+\beta+1)}
{(n+\alpha+1)(n+\beta+1)}.
}
\]
This includes the earlier simple region
\[
\alpha,\beta\ge2n,
\]
but it is noticeably stronger in asymmetric large-parameter regimes.

# 5. What remains

The full conjecture is **not yet honestly complete** from A2-4/A3-4 or from the present endpoint theorem.

After the new endpoint theorem, the remaining cases are much narrower. They are the two-sided overflow cases where at least one overflowing endpoint has exponent \(>1/2\) and the weighted-energy inequality above does not apply.

A typical unresolved configuration is:
\[
n\ge2,\qquad
0<\alpha\ll1,\qquad
\beta\approx(1.5\text{ to }2)n.
\]
The right endpoint is controlled by the new \(\alpha\le1/2\) theorem, but the left endpoint has exponent \(\beta>1/2\), and the elementary exponential Bessel envelope is too crude. Numerically, the sharper **true Bessel envelope**
\[
\max_{0\le\theta\le\Theta}
\left(\frac{\theta}{\sin\theta}\right)^{1/2}
J_\beta(M_L\theta)
\]
does appear to give enough slack, but proving that uniformly is the remaining step.

So the next exact lemma needed is:
\[
\boxed{
\text{Large-order endpoint Bessel-envelope lemma.}
}
\]

A usable form would be:

For \(a\ge1/2\), \(n\ge1\), and the endpoint-overflow admissible range,
\[
K_R
\max_{0\le\theta\le \Theta}
\left(\frac{\theta}{\sin\theta}\right)^{1/2}
J_a(M_R\theta)
\le
T_{n,a,b}.
\]

The crude bound
\[
J_a(z)
\le
\frac{(z/2)^a}{\Gamma(a+1)}e^{-z^2/(4(a+1))}
\]
is not strong enough in this sector. One needs either:
\[
\max J_a
\]
with sharp large-order control, or a direct Airy/Bessel transition estimate at the first maximum.

---

# Bottom line

A2-4 correctly identifies the architecture but falsely claims the proof is finished. A3-4 correctly identifies the remaining scalar endpoint problem but understates its difficulty.

The proof is now complete in a larger region than before, including all right endpoint overflows with \(\alpha\le1/2\), all left endpoint overflows with \(\beta\le1/2\), the central contour region, and the weighted-energy region. The only remaining obstruction is a sharp large-order endpoint Bessel-envelope lemma for overflowing endpoints with exponent \(>1/2\) outside the energy region.
