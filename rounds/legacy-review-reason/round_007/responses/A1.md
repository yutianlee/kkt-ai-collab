## 1. Evaluation of A2-6.md and A3-6.md

### A2-6.md

A2-6 is useful as a diagnostic document, but it does **not** complete the proof.

Its useful points are:
\[
\text{direct Sonin localization}
\quad+\quad
\text{central contour region}
\quad+\quad
\text{endpoint Bessel comparison}
\]
remain the right architecture. A2-6 correctly recognizes that the crude exponential Bessel majorant is too weak in the large-order endpoint regime; it explicitly says the fifth-round exponential envelope leaks above the KKT constant in some asymmetric examples.

The failure is its final "large-order" closure. A2-6 argues asymptotically that, for \(\alpha\gg\beta\),
\[
K_R\sim \left(\frac{2}{e}\right)^\alpha,
\]
so the Bessel envelope is crushed exponentially. This is not a proof for three reasons.

First, it is only an asymptotic statement, not a uniform inequality on the remaining compact transition region.

Second, the right-overflow regime \(x_+>\sigma\) is **not** generally a regime with \(\alpha\to\infty\) and fixed \(\beta\). In fact, the right-overflow condition forces an explicit algebraic restriction on \(\alpha\), derived below. For fixed \(n,\beta\), sufficiently large \(\alpha\) makes right overflow impossible.

Third, even in a large-parameter limit, the relevant hard limit is often \(\beta\to\infty\) with \(\alpha\) fixed or moderately sized, not \(\alpha\gg\beta\). In that limit the problem becomes a sharp Laguerre-type inequality, and A2-6 does not prove it.

So A2-6 is directionally useful but mathematically overclaims.

### A3-6.md

A3-6 is more structured, but its proof still has a gap.

It correctly identifies the four modules needed for a proof: Sonin localization, central contour region, weighted-energy region, and endpoint Bessel comparison. It also correctly states the direct Sonin functional
\[
S(x)=g(x)^2+\frac{A(x)^2g'(x)^2}{F(x)},
\qquad
S'(x)=-\frac{A(x)^2F'(x)}{F(x)^2}g'(x)^2,
\]
and uses this to reduce the maximum to the outer endpoint lobes.

The failure is again the final endpoint closure. A3-6 introduces a "Lemma A" asserting
\[
M_R^2>0,\qquad M_R^2\ge \frac14(n+1)^2
\]
under right overflow, but it gives only a verbal "direct algebraic verification" and says the detailed verification will be supplied in an appendix. More importantly, even if \(M_R^2>0\), positivity of the comparison frequency is far weaker than the required sharp inequality
\[
K_R
\sup_{0<\theta\le\Theta_R}
\left(\frac{\theta}{\sin\theta}\right)^{1/2}
J_\alpha(M_R\theta)
\le
T_{n,\alpha,\beta}.
\]

A3-6 uses a Landau-type Bessel maximum bound and then asserts that the remaining gamma-ratio inequality follows by routine monotonicity. That monotonicity is not proved in the file. So A3-6 is a good outline, but not a completed proof.

This matches the fifth-round diagnosis: the still-missing component is the large-order endpoint Bessel-envelope lemma.

# 2. What is solid

Let
\[
g(x)=g_n^{(\alpha,\beta)}(x)
=
C_{n,\alpha,\beta}
\left(\frac{1-x}{2}\right)^{\alpha/2}
\left(\frac{1+x}{2}\right)^{\beta/2}
P_n^{(\alpha,\beta)}(x),
\]
with
\[
C_{n,\alpha,\beta}^2
=
\frac{\Gamma(n+1)\Gamma(n+\alpha+\beta+1)}
{\Gamma(n+\alpha+1)\Gamma(n+\beta+1)}.
\]

Set
\[
A(x)=1-x^2,\qquad s=\alpha+\beta,\qquad d=\beta-\alpha,
\]
\[
\kappa=n(n+s+1)+\frac{s}{2}.
\]

Then
\[
\boxed{
(Ag')'+\frac{F(x)}{A(x)}g=0,
}
\]
where
\[
\boxed{
F(x)=\kappa(1-x^2)-\frac14(d-sx)^2.
}
\]

On \(F>0\),
\[
\boxed{
S_g(x)=g(x)^2+\frac{A(x)^2g'(x)^2}{F(x)}
}
\]
satisfies
\[
\boxed{
S_g'(x)=
-\frac{A(x)^2F'(x)}{F(x)^2}g'(x)^2.
}
\]

Thus all interior extrema are dominated by the outer endpoint lobes. This part is correct and should be kept. The same surviving structure is recorded in A1-6.

The turning points are
\[
\boxed{
x_\pm
=
\frac{ds\pm4\sqrt{\kappa(\kappa+\alpha\beta)}}
{s^2+4\kappa}.
}
\]

The central contour theorem also remains useful: if
\[
[x_-,x_+]\subseteq[-\sigma,\sigma],
\qquad
\sigma=\frac{s}{s+2n},
\]
then the refined KKT inequality follows from the contour estimate plus Sonin localization.

The weighted-energy region remains valid:
\[
|g(x)|^4
\le
\frac{s}{2\alpha\beta}
\left(
n+\frac12-\frac{1}{2(2n+s+1)}
\right),
\]
so the sharp KKT bound follows whenever
\[
\frac{s}{2\alpha\beta}
\left(
n+\frac12-\frac{1}{2(2n+s+1)}
\right)
\le
\frac{(n+1)(n+s+1)}
{(n+\alpha+1)(n+\beta+1)}.
\]

In particular, this includes
\[
\alpha,\beta\ge2n.
\]

The endpoint theorem for small endpoint exponent also remains valid:
\[
\text{right endpoint controlled if }0\le\alpha\le\frac12,
\]
and, by symmetry,
\[
\text{left endpoint controlled if }0\le\beta\le\frac12.
\]
A1-6 states this coverage explicitly.

# 3. New useful algebra: exact endpoint-overflow condition

The endpoint overflow can be written explicitly.

Let
\[
\sigma=\frac{s}{s+2n}.
\]

A direct calculation gives
\[
\boxed{
F(\sigma)
=
\frac{\Phi_R(n,\alpha,\beta)}{s+2n},
}
\]
where
\[
\boxed{
\Phi_R
=
2n^3+2n^2
+
(3n^2+2n)(\alpha+\beta)
+
2n\alpha\beta
-
\alpha^2(\alpha+\beta).
}
\]

Since \(F(1)=-\alpha^2\le0\), the right endpoint overflows the central region precisely when
\[
\boxed{
x_+>\sigma
\quad\Longleftrightarrow\quad
\Phi_R>0.
}
\]

Similarly,
\[
\boxed{
F(-\sigma)
=
\frac{\Phi_L(n,\alpha,\beta)}{s+2n},
}
\]
where
\[
\boxed{
\Phi_L
=
2n^3+2n^2
+
(3n^2+2n)(\alpha+\beta)
+
2n\alpha\beta
-
\beta^2(\alpha+\beta).
}
\]

Thus
\[
\boxed{
x_-<-\sigma
\quad\Longleftrightarrow\quad
\Phi_L>0.
}
\]

This is useful because it shows that A2-6's "\(\alpha\gg\beta\)" right-overflow closure is aimed at the wrong asymptotic. For fixed \(n,\beta\),
\[
\Phi_R
\sim -\alpha^3
\qquad(\alpha\to\infty),
\]
so right overflow eventually disappears. The hard right-overflow regime is not \(\alpha\to\infty\) with fixed \(\beta\). It is more like
\[
\beta\to\infty,\qquad \alpha \text{ fixed or moderately sized}.
\]

Indeed, as \(\beta\to\infty\),
\[
\Phi_R
=
\beta\bigl(3n^2+2n+2n\alpha-\alpha^2\bigr)
+O(1).
\]
So right overflow persists in the large-\(\beta\) limit exactly when
\[
\boxed{
\alpha^2-2n\alpha-3n^2-2n<0,
}
\]
i.e.
\[
\boxed{
\alpha<n+\sqrt{4n^2+2n}.
}
\]

The genuinely unresolved sector includes
\[
\boxed{
\frac12<\alpha<2n,\qquad \beta\to\infty,
}
\]
with the left/right roles swapped for the other endpoint. This is outside the small-exponent theorem and outside the balanced-energy theorem.

# 4. The large-\(\beta\) endpoint limit: a necessary Laguerre inequality

This is the cleanest way to expose the remaining obstruction.

Fix \(n\ge1\), \(\alpha\ge0\), and set
\[
x=1-\frac{2u}{\beta},
\qquad u\ge0.
\]

As \(\beta\to\infty\),
\[
\left(\frac{1-x}{2}\right)^{\alpha/2}
=
\left(\frac{u}{\beta}\right)^{\alpha/2},
\]
\[
\left(\frac{1+x}{2}\right)^{\beta/2}
=
\left(1-\frac{u}{\beta}\right)^{\beta/2}
\to e^{-u/2},
\]
and the standard Jacobi-to-Laguerre limit gives
\[
P_n^{(\alpha,\beta)}
\left(1-\frac{2u}{\beta}\right)
\to
L_n^{(\alpha)}(u).
\]

Also,
\[
C_{n,\alpha,\beta}^2
=
\frac{\Gamma(n+1)\Gamma(n+\alpha+\beta+1)}
{\Gamma(n+\alpha+1)\Gamma(n+\beta+1)}
\sim
\frac{\Gamma(n+1)}{\Gamma(n+\alpha+1)}\beta^\alpha.
\]

Therefore
\[
\boxed{
g_n^{(\alpha,\beta)}
\left(1-\frac{2u}{\beta}\right)
\to
\ell_n^{(\alpha)}(u)
}
\]
locally uniformly in \(u\), where
\[
\boxed{
\ell_n^{(\alpha)}(u)
=
\left(
\frac{\Gamma(n+1)}
{\Gamma(n+\alpha+1)}
\right)^{1/2}
u^{\alpha/2}e^{-u/2}
L_n^{(\alpha)}(u).
}
\]

Meanwhile,
\[
T_{n,\alpha,\beta}^4
=
\frac{(n+1)(n+\alpha+\beta+1)}
{(n+\alpha+1)(n+\beta+1)}
\to
\frac{n+1}{n+\alpha+1}.
\]

Thus the KKT conjecture implies the sharp Laguerre endpoint inequality
\[
\boxed{
\left|
\left(
\frac{\Gamma(n+1)}
{\Gamma(n+\alpha+1)}
\right)^{1/2}
u^{\alpha/2}e^{-u/2}
L_n^{(\alpha)}(u)
\right|
\le
\left(
\frac{n+1}{n+\alpha+1}
\right)^{1/4}
}
\tag{LAG}
\]
for all
\[
u\ge0,\qquad \alpha\ge0,\qquad n\ge1.
\]

This is not a side issue. It is the exact endpoint blow-up limit of the remaining right-overflow sector. Any complete Sonin/Bessel proof must prove either \((\mathrm{LAG})\) or a strictly stronger finite-\(\beta\) endpoint estimate that implies it.

Neither A2-6 nor A3-6 proves \((\mathrm{LAG})\).

# 5. What a completion would now require

The proof architecture is now reduced to one sharp endpoint lemma.

For the right endpoint, define
\[
x=\cos\theta,
\qquad
v(\theta)=\sqrt{\sin\theta}\,g(\cos\theta).
\]

Then
\[
v''+Q(\theta)v=0,
\]
where
\[
Q(\theta)
=
N^2
+
\frac{1/4-\alpha^2}{4\sin^2(\theta/2)}
+
\frac{1/4-\beta^2}{4\cos^2(\theta/2)},
\qquad
N=n+\frac{\alpha+\beta+1}{2}.
\]

With
\[
\Theta_R=\arccos\frac{\alpha+\beta}{\alpha+\beta+2n},
\]
one has the comparison envelope
\[
\boxed{
|g(\cos\theta)|
\le
K_R
\left(\frac{\theta}{\sin\theta}\right)^{1/2}
J_\alpha(M_R\theta)
}
\]
on the right endpoint lobe, where
\[
K_R
=
C_{n,\alpha,\beta}
\frac{\Gamma(n+\alpha+1)}
{\Gamma(n+1)M_R^\alpha}.
\]

The missing lemma is exactly
\[
\boxed{
K_R
\sup_{0<\theta\le\Theta_R}
\left(\frac{\theta}{\sin\theta}\right)^{1/2}
J_\alpha(M_R\theta)
\le
T_{n,\alpha,\beta}
}
\tag{BES}
\]
for the remaining range
\[
\alpha>\frac12
\]
outside the central and energy regions. The left endpoint version is the same with
\[
\alpha\leftrightarrow\beta.
\]

A3-6 tries to prove this by replacing the Bessel factor with a global Landau bound and then claiming the rest is monotone. But the monotonicity and gamma-ratio comparison are not actually supplied. A2-6 tries to prove it by a large-\(\alpha\) asymptotic decay of \(K_R\), but the relevant endpoint-overflow limit includes \(\beta\to\infty\) with fixed/moderate \(\alpha\), where the Laguerre limit \((\mathrm{LAG})\) is the actual model.

# 6. Current best proof status

The refined KKT inequality is rigorously covered by the current Sonin/Bessel program in these regimes:
\[
\boxed{
[x_-,x_+]\subseteq[-\sigma,\sigma]
}
\]
by central contour plus Sonin localization;

\[
\boxed{
0\le\alpha\le\frac12
}
\]
for the right endpoint lobe, and
\[
\boxed{
0\le\beta\le\frac12
}
\]
for the left endpoint lobe;

\[
\boxed{
\frac{s}{2\alpha\beta}
\left(
n+\frac12-\frac{1}{2(2n+s+1)}
\right)
\le
\frac{(n+1)(n+s+1)}
{(n+\alpha+1)(n+\beta+1)}
}
\]
by the weighted-energy method, in particular
\[
\boxed{
\alpha,\beta\ge2n.
}
\]

The remaining problem is not vague anymore. It is the sharp finite-\(\beta\) endpoint Bessel inequality \((\mathrm{BES})\), whose limiting form is the sharp Laguerre inequality \((\mathrm{LAG})\).

I would not mark the KKT conjecture as proved from A2-6 or A3-6. The best honest next theorem to target is:
\[
\boxed{
\text{Prove }(\mathrm{LAG})\text{ for all }n\ge1,\ \alpha\ge0,
\text{ then lift it to finite }\beta\text{ via Sturm comparison.}
}
\]

That is the cleanest remaining route to a complete analytic proof without integer-parameter representation theory.
