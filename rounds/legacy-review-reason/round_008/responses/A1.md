## 1. Evaluation of A2-7, A3-7, A4-7

### A2-7

A2-7 correctly identifies one useful fact: right endpoint overflow cannot persist for arbitrarily large \(\alpha\) at fixed \(n,\beta\). It states the bound
\[
x_+>\sigma
\quad\Longrightarrow\quad
\alpha<n+\sqrt{4n^2+2n}.
\]

That structural observation is useful. But A2-7 then overclaims. It says this makes the large-order gap "fictitious" and that the Hadamard/Wendel-Gautschi estimates "apply seamlessly" to close
\[
E_R\le T_{n,\alpha,\beta}.
\]
No such inequality chain is supplied. A2-7 explicitly jumps from "overflow is bounded" to "the exponential majorant flawlessly covers the finite remaining space." That is not a proof.

The deeper issue: the bound
\[
\alpha<n+\sqrt{4n^2+2n}\sim 3n
\]
is not a compact bound independent of \(n\). It still leaves an unbounded two-parameter transition region. Also, for fixed moderate \(\alpha\), right overflow can persist as \(\beta\to\infty\), which leads to a genuine Laguerre endpoint problem, not a finite leftover check.

### A3-7

A3-7 has a valid setup through the direct ODE, Sonin localization, central contour region, and a Bessel comparison. But its final scalar closure fails.

The critical step is its use of
\[
|J_\alpha(z)|\le \frac{(z/2)^\alpha}{\Gamma(\alpha+1)}
\]
to cancel \(M_R\), yielding the scalar bound labelled (12). A3-7 then says the remaining inequality is handled by a "routine but lengthy monotonicity argument" deferred to an appendix, and concludes the proof.

That scalar inequality is not true as a uniform sufficient bound. For example, with
\[
n=1,\qquad \alpha=0.5170920243,\qquad \beta=0.001,
\]
one has right overflow
\[
x_+\approx0.9529955>\sigma\approx0.2057479,
\]
and the energy estimate fails. But A3-7's scalar upper bound gives
\[
\frac{\text{A3-7 bound (12)}}{T_{n,\alpha,\beta}}\approx 1.75428>1.
\]
So A3-7's proposed closure cannot prove the KKT inequality. Numerically, the actual normalized Jacobi maximum is still below the KKT target:
\[
\frac{\max_{[-1,1]}|g_n^{(\alpha,\beta)}|}{T_{n,\alpha,\beta}}
\approx 0.99637,
\]
but A3-7's bound is too crude to certify it.

### A4-7

A4-7 is the most useful response. It correctly identifies the shared architecture:

1. direct Sonin-Polya localization;
2. central branch-safe contour region;
3. small endpoint exponent theorem;
4. weighted-energy region;
5. one remaining large-order endpoint-lobe problem.

It explicitly marks the large-order endpoint lobe as the sole remaining obstruction.

It also correctly explains why Landau's global Bessel bound alone is too loose near \(\alpha=1/2\), and proposes the right kind of remaining work: use sharper Bessel maximum bounds, explicit Stirling estimates for \(K_R\), and a rigorous large-\(n\)/finite-interval verification split.

So A4-7 should be kept as the roadmap. It does not prove the conjecture, but it does not hallucinate the final step.

## 2. Validated core of the proof architecture

Let
\[
g(x)=g_n^{(\alpha,\beta)}(x)
=
C_{n,\alpha,\beta}
\left(\frac{1-x}{2}\right)^{\alpha/2}
\left(\frac{1+x}{2}\right)^{\beta/2}
P_n^{(\alpha,\beta)}(x),
\]
where
\[
C_{n,\alpha,\beta}^2
=
\frac{\Gamma(n+1)\Gamma(n+\alpha+\beta+1)}
{\Gamma(n+\alpha+1)\Gamma(n+\beta+1)}.
\]

The original KKT paper states the refined estimate for integer parameters and conjectures its validity for all real \(\alpha,\beta\ge0\). ([Pure][1]) The uploaded review says the same: the refined estimate is proven for integer parameters, while the full real-parameter version is the conjectural target.

Set
\[
A(x)=1-x^2,\qquad s=\alpha+\beta,\qquad d=\beta-\alpha,
\]
\[
\kappa=n(n+s+1)+\frac{s}{2}.
\]

Then
\[
\boxed{
(Ag')'+\frac{F(x)}{A(x)}g=0,
}
\]
with
\[
\boxed{
F(x)
=
\kappa(1-x^2)-\frac14(d-sx)^2.
}
\]

On \(F>0\), define
\[
S_g(x)
=
g(x)^2+\frac{A(x)^2g'(x)^2}{F(x)}.
\]
A direct calculation gives
\[
\boxed{
S_g'(x)
=
-\frac{A(x)^2F'(x)}{F(x)^2}g'(x)^2.
}
\]

This is the correct Sonin functional. Since \(F\) is a concave quadratic, \(S_g\) decreases from the left turning point to the vertex of \(F\), then increases from the vertex to the right turning point. Thus the largest interior extrema are forced toward the two outer endpoint lobes. This part is solid and is also recognized in the previous files.

The central contour theorem also remains valid as a conditional module: if
\[
\sigma=\frac{\alpha+\beta}{\alpha+\beta+2n}
\]
and
\[
[x_-,x_+]\subseteq[-\sigma,\sigma],
\]
then the sharp KKT bound follows from the branch-safe Haagerup-Schlichtkrull contour estimate plus Sonin localization.

The weighted-energy region remains valid: for \(\alpha,\beta>0\),
\[
|g(x)|^4
\le
\frac{s}{2\alpha\beta}
\left(
n+\frac12-\frac{1}{2(2n+s+1)}
\right),
\]
so the refined KKT inequality follows whenever
\[
\frac{s}{2\alpha\beta}
\left(
n+\frac12-\frac{1}{2(2n+s+1)}
\right)
\le
\frac{(n+1)(n+s+1)}
{(n+\alpha+1)(n+\beta+1)}.
\]
In particular, this covers \(\alpha,\beta\ge2n\).

## 3. New exact overflow calculation

The seventh-round files use overflow language, but the exact algebra is worth recording cleanly.

Let
\[
\sigma=\frac{s}{s+2n}.
\]

A direct substitution into \(F\) gives
\[
\boxed{
F(\sigma)=\frac{\Phi_R(n,\alpha,\beta)}{s+2n},
}
\]
where
\[
\boxed{
\Phi_R
=
2n^3+2n^2
+
(3n^2+2n)(\alpha+\beta)
+
2n\alpha\beta
-
\alpha^2(\alpha+\beta).
}
\]

Since \(F(1)=-\alpha^2\le0\), right overflow is equivalent to
\[
\boxed{
x_+>\sigma
\quad\Longleftrightarrow\quad
\Phi_R>0.
}
\]

Similarly,
\[
\boxed{
F(-\sigma)=\frac{\Phi_L(n,\alpha,\beta)}{s+2n},
}
\]
where
\[
\boxed{
\Phi_L
=
2n^3+2n^2
+
(3n^2+2n)(\alpha+\beta)
+
2n\alpha\beta
-
\beta^2(\alpha+\beta),
}
\]
and
\[
\boxed{
x_-<-\sigma
\quad\Longleftrightarrow\quad
\Phi_L>0.
}
\]

Now view \(\Phi_R\) as a function of \(\beta\):
\[
\Phi_R
=
\Bigl[
2n^3+2n^2+(3n^2+2n)\alpha-\alpha^3
\Bigr]
+
\beta\Bigl[
3n^2+2n+2n\alpha-\alpha^2
\Bigr].
\]

If
\[
\alpha\ge n+\sqrt{4n^2+2n},
\]
then
\[
3n^2+2n+2n\alpha-\alpha^2\le0.
\]
Moreover, the constant term is already negative at that threshold and decreases thereafter. Hence
\[
\Phi_R\le0.
\]
Therefore
\[
\boxed{
x_+>\sigma
\quad\Longrightarrow\quad
\alpha<n+\sqrt{4n^2+2n}.
}
\]

Symmetrically,
\[
\boxed{
x_-<-\sigma
\quad\Longrightarrow\quad
\beta<n+\sqrt{4n^2+2n}.
}
\]

This validates the useful part of A2-7. But it does **not** complete the proof, because the threshold still grows like \(3n\), and because one can still have \(\beta\to\infty\) with fixed moderate \(\alpha\).

## 4. The real remaining obstruction: a Laguerre endpoint inequality

The sharp way to see the unresolved endpoint regime is to take the right-endpoint limit
\[
\beta\to\infty,\qquad
x=1-\frac{2u}{\beta},
\qquad u\ge0,
\]
with \(n,\alpha\) fixed.

Then
\[
\left(\frac{1-x}{2}\right)^{\alpha/2}
=
\left(\frac{u}{\beta}\right)^{\alpha/2},
\]
\[
\left(\frac{1+x}{2}\right)^{\beta/2}
=
\left(1-\frac{u}{\beta}\right)^{\beta/2}
\to e^{-u/2},
\]
and the Jacobi-to-Laguerre limit gives
\[
P_n^{(\alpha,\beta)}
\left(1-\frac{2u}{\beta}\right)
\to
L_n^{(\alpha)}(u).
\]

The normalization satisfies
\[
C_{n,\alpha,\beta}^2
\sim
\frac{\Gamma(n+1)}{\Gamma(n+\alpha+1)}
\beta^\alpha.
\]

Therefore
\[
g_n^{(\alpha,\beta)}
\left(1-\frac{2u}{\beta}\right)
\to
\ell_n^{(\alpha)}(u),
\]
where
\[
\boxed{
\ell_n^{(\alpha)}(u)
=
\left(
\frac{\Gamma(n+1)}
{\Gamma(n+\alpha+1)}
\right)^{1/2}
u^{\alpha/2}e^{-u/2}L_n^{(\alpha)}(u).
}
\]

Meanwhile,
\[
T_{n,\alpha,\beta}^4
\to
\frac{n+1}{n+\alpha+1}.
\]

Thus any complete proof of the KKT conjecture must prove the sharp Laguerre endpoint inequality
\[
\boxed{
|\ell_n^{(\alpha)}(u)|
\le
\left(
\frac{n+1}{n+\alpha+1}
\right)^{1/4}
\qquad
(u\ge0,\ \alpha\ge0,\ n\ge1).
}
\tag{LAG}
\]

This is exactly the obstruction identified in the earlier honest reduction.

A2-7 does not prove this. A3-7 does not prove this. A4-7 correctly points toward sharper Bessel/Stirling estimates as the needed mechanism.

## 5. A new validated partial proof for the Laguerre limit

The Laguerre limit itself has a useful energy proof in the large-\(\alpha\) range.

Let
\[
y(u)=\ell_n^{(\alpha)}(u).
\]

Then \(y\) satisfies
\[
\boxed{
(uy')'
+
\left(
n+\frac{\alpha+1}{2}
-\frac{u}{4}
-\frac{\alpha^2}{4u}
\right)y
=0.
}
\tag{LagODE}
\]

The standard Laguerre norm identities give
\[
\int_0^\infty y(u)^2\,du=1,
\]
\[
\int_0^\infty \frac{y(u)^2}{u}\,du=\frac1\alpha
\qquad(\alpha>0),
\]
and
\[
\int_0^\infty u\,y'(u)^2\,du
=
\frac{2n+1}{4}.
\]

For any \(u>0\),
\[
y(u)^2
=
2\int_0^u y(t)y'(t)\,dt
=
-2\int_u^\infty y(t)y'(t)\,dt.
\]

Using Cauchy-Schwarz with weights \(t^{-1}\) and \(t\), and splitting at \(u\), gives
\[
|y(u)|^4
\le
\left(\int_0^\infty \frac{y(t)^2}{t}\,dt\right)
\left(\int_0^\infty t\,y'(t)^2\,dt\right)
=
\frac{2n+1}{4\alpha}.
\]

Thus
\[
\boxed{
|\ell_n^{(\alpha)}(u)|^4
\le
\frac{2n+1}{4\alpha}.
}
\]

This implies the sharp Laguerre target whenever
\[
\frac{2n+1}{4\alpha}
\le
\frac{n+1}{n+\alpha+1}.
\]
Equivalently,
\[
\boxed{
\alpha
\ge
\frac{(2n+1)(n+1)}{2n+3}.
}
\tag{LagEnergy}
\]

So the Laguerre endpoint inequality \((\mathrm{LAG})\) is now proved in the large-\(\alpha\) range
\[
\boxed{
\alpha
\ge
\frac{(2n+1)(n+1)}{2n+3}
\sim n.
}
\]

The previously established small endpoint theorem covers \(0\le\alpha\le1/2\). Therefore, in the Laguerre blow-up limit, the unresolved strip is reduced to
\[
\boxed{
\frac12<\alpha<
\frac{(2n+1)(n+1)}{2n+3}.
}
\]

This is a genuine narrowing, but not a full proof.

## 7. Final status

The best validated proof status is:
\[
\boxed{
\text{The full KKT conjecture is not yet proved by A2-7, A3-7, or A4-7.}
}
\]

What is proved or conditionally secured:
\[
\boxed{
[x_-,x_+]\subseteq[-\sigma,\sigma]
}
\]
by central contour plus Sonin localization;

\[
\boxed{
0\le\alpha\le\frac12
}
\]
for the right endpoint lobe, and
\[
\boxed{
0\le\beta\le\frac12
}
\]
for the left endpoint lobe;

\[
\boxed{
\frac{s}{2\alpha\beta}
\left(
n+\frac12-\frac{1}{2(2n+s+1)}
\right)
\le
\frac{(n+1)(n+s+1)}
{(n+\alpha+1)(n+\beta+1)}
}
\]
by weighted energy, including \(\alpha,\beta\ge2n\);

and, in the Laguerre endpoint limit,
\[
\boxed{
\alpha
\ge
\frac{(2n+1)(n+1)}{2n+3}
}
\]
by the new Laguerre energy estimate.

The remaining analytic task is now sharply formulated:
\[
\boxed{
\text{Prove the sharp Laguerre inequality }(\mathrm{LAG})
\text{ in the strip }
\frac12<\alpha<
\frac{(2n+1)(n+1)}{2n+3},
}
\]
then lift it to finite \(\beta\) using the endpoint Sturm comparison. A4-7's proposed direction, sharp Bessel maximum estimates plus explicit Stirling bounds, is the most credible route for that final step.
