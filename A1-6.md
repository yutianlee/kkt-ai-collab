## 1. Evaluation of A2-5.md and A3-5.md

**A2-5.md does not complete the proof.** It correctly reuses the direct Sonin localization and the branch-safe Haagerup-Schlichtkrull central contour idea, but its final endpoint closure is not an inequality proof. The file proves, or at least plausibly argues, that the shifted endpoint comparison frequency \(M_R^2\) remains positive in extreme asymmetric regimes; that is useful. But positivity of \(M_R^2\) only says the Bessel comparison model is not imaginary. It does **not** prove the required sharp scalar bound
\[
K_R\sup_{0<\theta\le\Theta_R}
\left(\frac{\theta}{\sin\theta}\right)^{1/2}
J_\alpha(M_R\theta)
\le
T_{n,\alpha,\beta}.
\]
A2-5 ends with verbal assertions that the "Bessel envelope natively absorbs the scaling constants" and that the KKT \(1/4\)-power slack absorbs constants, but gives no verifiable chain of inequalities.

**A3-5.md is more structured, but its endpoint closure is invalid as written.** It correctly identifies the remaining scalar endpoint task \(E_R\le T\), and it sets up the Bessel envelope
\[
E_R
=
D_R
\left(\frac{\alpha}{2e c_R}\right)^{\alpha/2}.
\]
But then it closes the proof by an unproved monotonicity claim
\[
\partial_\alpha(\log E_R-\log T)<0
\]
and refers to an Appendix that is not actually present in the retrieved file. More seriously, A3-5 uses the bound
\[
\rho(\Theta_R)\le \frac1{12},
\qquad
\rho(\Theta)=\sup_{0<t\le\Theta}
\frac{1}{2t^2}\log\frac{t}{\sin t},
\]
but this is false for every positive \(\Theta\). Indeed
\[
\log\frac{t}{\sin t}
=
\frac{t^2}{6}+\frac{t^4}{180}+O(t^6),
\]
so
\[
\rho(\Theta)
=
\frac1{12}+\frac{\Theta^2}{360}+O(\Theta^4)
>
\frac1{12}.
\]
At \(\Theta=\pi/2\),
\[
\rho(\pi/2)
=
\frac{2}{\pi^2}\log\frac{\pi}{2}
\approx 0.0914
>
0.0833=\frac1{12}.
\]
Thus the displayed lower bound
\[
c_R\ge \frac{M_R^2}{4(\alpha+1)}-\frac1{12}
\]
has the wrong direction in general. A3-5's scalar endpoint proof therefore cannot be accepted.

So the useful content is:
\[
\boxed{
\text{A2-5: useful for }M_R^2\text{ positivity intuition, not endpoint constants.}
}
\]
\[
\boxed{
\text{A3-5: useful for organizing the endpoint scalar problem, but the closure fails.}
}
\]

The uploaded review and the original KKT paper remain consistent with this: KKT's refined estimate is proven for integer parameters, while the full real-parameter statement is the conjectural target. The original KKT paper states Conjecture 6.1 in exactly this form. ([pure.uva.nl][1])

# 2. What remains valid

Let
\[
g(x)=g_n^{(\alpha,\beta)}(x)
=
C_{n,\alpha,\beta}
\left(\frac{1-x}{2}\right)^{\alpha/2}
\left(\frac{1+x}{2}\right)^{\beta/2}
P_n^{(\alpha,\beta)}(x),
\]
where
\[
C_{n,\alpha,\beta}^2
=
\frac{\Gamma(n+1)\Gamma(n+\alpha+\beta+1)}
{\Gamma(n+\alpha+1)\Gamma(n+\beta+1)}.
\]

Set
\[
A(x)=1-x^2,\qquad s=\alpha+\beta,\qquad d=\beta-\alpha,
\]
\[
N=n+\frac{s+1}{2},
\qquad
\kappa=n(n+s+1)+\frac{s}{2}.
\]

Then
\[
\boxed{
(Ag')'+\frac{F(x)}{A(x)}g=0,
}
\]
where
\[
\boxed{
F(x)
=
\kappa(1-x^2)
-\frac14(d-sx)^2.
}
\]

On \(F>0\), the direct Sonin functional
\[
\boxed{
S_g(x)=g(x)^2+\frac{A(x)^2g'(x)^2}{F(x)}
}
\]
satisfies
\[
\boxed{
S_g'(x)
=
-\frac{A(x)^2F'(x)}{F(x)^2}g'(x)^2.
}
\]
This part is solid and appears consistently across the better attempts.

The turning points are
\[
\boxed{
x_\pm
=
\frac{ds\pm4\sqrt{\kappa(\kappa+\alpha\beta)}}
{s^2+4\kappa}.
}
\]
For \(\alpha,\beta>0\),
\[
F(x)>0
\quad\Longleftrightarrow\quad
x\in(x_-,x_+).
\]

Let
\[
\sigma=\frac{s}{s+2n}.
\]
The central contour theorem says that the refined KKT bound holds on
\[
|x|<\sigma.
\]
Therefore, if
\[
[x_-,x_+]\subseteq[-\sigma,\sigma],
\]
then the full inequality follows by Sonin localization. This central-region reduction is valid and useful.

The unresolved part remains the endpoint overflow:
\[
x_+>\sigma
\quad\text{or}\quad
x_-<-\sigma.
\]

# 3. A corrected endpoint theorem that genuinely advances the proof

The fifth-round attempts fail for large endpoint exponents, but the endpoint Bessel method **does** close an important region.

## Theorem: right endpoint closed when \(0\le\alpha\le\frac12\)

For all
\[
n\ge1,\qquad 0\le\alpha\le\frac12,\qquad \beta\ge0,
\]
the right endpoint lobe satisfies
\[
\boxed{
|g_n^{(\alpha,\beta)}(x)|
\le
T_{n,\alpha,\beta}
}
\]
where
\[
T_{n,\alpha,\beta}
=
\left(
\frac{(n+1)(n+\alpha+\beta+1)}
{(n+\alpha+1)(n+\beta+1)}
\right)^{1/4}.
\]

By symmetry, the left endpoint lobe is closed when
\[
0\le\beta\le\frac12.
\]

### Proof

Put
\[
x=\cos\theta,
\qquad
v(\theta)=\sqrt{\sin\theta}\,g(\cos\theta).
\]
Then
\[
v''+Q(\theta)v=0,
\]
where
\[
Q(\theta)
=
N^2
+
\frac{1/4-\alpha^2}{4\sin^2(\theta/2)}
+
\frac{1/4-\beta^2}{4\cos^2(\theta/2)}.
\]

Let
\[
\Theta=\arccos\sigma,
\qquad
\sigma=\frac{\alpha+\beta}{\alpha+\beta+2n}.
\]
Then \(0<\Theta\le\pi/2\) unless \(s=0\).

Define
\[
B(\theta)=\frac1{4\sin^2(\theta/2)}-\frac1{\theta^2},
\qquad
C(\theta)=\frac1{4\cos^2(\theta/2)}.
\]
On \(0<\theta\le\Theta\), both \(B\) and \(C\) are increasing.

For \(0\le\alpha\le1/2\), set
\[
M_R^2
=
N^2+\frac{1/4-\alpha^2}{12}+m_\beta,
\]
where
\[
m_\beta
=
\begin{cases}
\dfrac{1/4-\beta^2}{4},
&0\le\beta\le\frac12,\\[1.2ex]
\left(\dfrac14-\beta^2\right)
\dfrac{1}{4\cos^2(\Theta/2)},
&\beta>\frac12.
\end{cases}
\]
Then
\[
\boxed{
Q(\theta)
\ge
M_R^2+\frac{1/4-\alpha^2}{\theta^2}
\qquad(0<\theta\le\Theta).
}
\]

The regular comparison solution is
\[
z(\theta)=K_R\sqrt{\theta}J_\alpha(M_R\theta),
\]
with
\[
\boxed{
K_R
=
C_{n,\alpha,\beta}
\frac{\Gamma(n+\alpha+1)}
{\Gamma(n+1)M_R^\alpha}.
}
\]

Singular Sturm comparison gives, through the right endpoint lobe,
\[
0\le g(\cos\theta)
\le
K_R
\left(\frac{\theta}{\sin\theta}\right)^{1/2}
J_\alpha(M_R\theta).
\]

Use the Hadamard-product Bessel estimate
\[
J_\alpha(z)
\le
\frac{(z/2)^\alpha}{\Gamma(\alpha+1)}
\exp\left(-\frac{z^2}{4(\alpha+1)}\right),
\qquad 0\le z\le j_{\alpha,1}.
\]
Also use the corrected elementary bound
\[
\left(\frac{\theta}{\sin\theta}\right)^{1/2}
\le
\exp\left(\frac{3\theta^2}{32}\right),
\qquad 0\le\theta\le\frac\pi2.
\]
Thus
\[
g(\cos\theta)
\le
D_R\theta^\alpha e^{-c_R\theta^2},
\]
where
\[
D_R
=
\frac{
C_{n,\alpha,\beta}\Gamma(n+\alpha+1)
}
{
\Gamma(n+1)2^\alpha\Gamma(\alpha+1)
},
\]
and
\[
c_R=
\frac{M_R^2}{4(\alpha+1)}-\frac{3}{32}.
\]

A direct algebraic estimate gives
\[
\boxed{
c_R
\ge
\frac{(n+1)(n+\beta+1)}{4e}.
}
\tag{1}
\]
Here is the verification. Since \(e>8/3\), it is enough to prove
\[
8M_R^2
\ge
3(\alpha+1)\bigl((n+1)(n+\beta+1)+1\bigr).
\]
Put \(m=n+1\ge2\).

For \(0\le\beta\le1/2\), the difference equals
\[
\frac{
4\alpha^2-9\alpha\beta m+12\alpha\beta
-9\alpha m^2+24\alpha m-21\alpha
+15\beta m-12\beta
+15m^2-24m-1
}{3}.
\]
This expression is increasing in \(\beta\) and decreasing in \(\alpha\) on
\[
0\le\alpha,\beta\le\frac12,\qquad m\ge2.
\]
Hence it is minimized at \(\alpha=1/2,\beta=0\), where it equals
\[
\frac{7m^2-8m-7}{2}>0.
\]

For \(\beta\ge1/2\), after multiplying the same difference by the positive denominator
\[
6(\alpha+\beta+m-1),
\]
one obtains a quadratic polynomial in \(\beta\), increasing for \(\beta\ge1/2\). At \(\beta=1/2\), it factors as
\[
\frac{2\alpha+2m-1}{2}
\left(
8\alpha^2-18\alpha m^2+39\alpha m-30\alpha
+30m^2-33m-14
\right).
\]
The second factor is decreasing in \(\alpha\in[0,1/2]\), so its minimum occurs at \(\alpha=1/2\), where it becomes
\[
\frac32(2m-3)(7m+6)>0.
\]
Thus \((1)\) follows.

Now optimize
\[
\theta^\alpha e^{-c_R\theta^2}.
\]
For \(\alpha>0\),
\[
\max_{\theta\ge0}\theta^\alpha e^{-c_R\theta^2}
=
\left(\frac{\alpha}{2ec_R}\right)^{\alpha/2},
\]
with value \(1\) at \(\alpha=0\). Therefore
\[
|g(\cos\theta)|
\le
D_R
\left(\frac{\alpha}{2ec_R}\right)^{\alpha/2}.
\]

By Wendel/Gautschi gamma-ratio inequalities,
\[
\frac{\Gamma(n+\alpha+1)}{\Gamma(n+1)}
\le
(n+1)^\alpha,
\]
and
\[
\frac{\Gamma(n+\alpha+\beta+1)}
{\Gamma(n+\beta+1)}
\le
(n+\beta+1)^\alpha.
\]
Hence
\[
D_R
\le
\frac{
\bigl((n+1)(n+\beta+1)\bigr)^{\alpha/2}
}
{
2^\alpha\Gamma(\alpha+1)
}.
\]
Using \((1)\),
\[
D_R
\left(\frac{\alpha}{2ec_R}\right)^{\alpha/2}
\le
\frac1{\Gamma(\alpha+1)}
\left(\frac{\alpha}{2}\right)^{\alpha/2}.
\]

Finally, for \(0\le\alpha\le1/2\),
\[
\frac1{\Gamma(\alpha+1)}
\left(\frac{\alpha}{2}\right)^{\alpha/2}
\le
\left(\frac{2}{2+\alpha}\right)^{1/4}.
\]
Also,
\[
T_{n,\alpha,\beta}^4
=
1-\frac{\alpha\beta}
{(n+\alpha+1)(n+\beta+1)}
\ge
1-\frac{\alpha}{n+\alpha+1}
=
\frac{n+1}{n+\alpha+1}
\ge
\frac{2}{2+\alpha}.
\]
Thus
\[
|g(\cos\theta)|\le T_{n,\alpha,\beta}.
\]
This proves the right endpoint theorem.

The left endpoint theorem follows by swapping
\[
\alpha\leftrightarrow\beta,\qquad x\leftrightarrow -x.
\]

# 4. Current rigorous proof coverage

The refined KKT inequality is now rigorously covered in the following regimes.

### A. Central oscillatory regime

If
\[
[x_-,x_+]\subseteq[-\sigma,\sigma],
\qquad
\sigma=\frac{\alpha+\beta}{\alpha+\beta+2n},
\]
then the contour theorem plus direct Sonin localization proves the result.

### B. Small endpoint exponent overflow

If the right endpoint overflows but
\[
0\le\alpha\le\frac12,
\]
the theorem above controls the right lobe.

If the left endpoint overflows but
\[
0\le\beta\le\frac12,
\]
the symmetric theorem controls the left lobe.

### C. Weighted-energy region

For \(\alpha,\beta>0\), the weighted-energy estimate gives
\[
|g(x)|^4
\le
\frac{\alpha+\beta}{2\alpha\beta}
\left(
n+\frac12-\frac{1}{2(2n+\alpha+\beta+1)}
\right).
\]
Therefore the refined KKT inequality follows whenever
\[
\frac{\alpha+\beta}{2\alpha\beta}
\left(
n+\frac12-\frac{1}{2(2n+\alpha+\beta+1)}
\right)
\le
\frac{(n+1)(n+\alpha+\beta+1)}
{(n+\alpha+1)(n+\beta+1)}.
\]
This includes, in particular,
\[
\alpha,\beta\ge2n.
\]

# 5. What is still missing for the full conjecture

The only remaining obstruction is the **large-order endpoint lobe**.

The exact missing lemma is:
\[
\boxed{
\text{Large-order endpoint Bessel-envelope lemma.}
}
\]

For the right endpoint, with \(\alpha>1/2\), prove
\[
\boxed{
K_R
\sup_{0<\theta\le\Theta_R}
\left(\frac{\theta}{\sin\theta}\right)^{1/2}
J_\alpha(M_R\theta)
\le
T_{n,\alpha,\beta}.
}
\tag{L}
\]
The left endpoint version is the same with \(\alpha,\beta\) interchanged.

A2-5 does not prove \((L)\); it only argues that \(M_R^2>0\). A3-5 tries to replace \((L)\) by the crude exponential envelope \(E_R\le T\), but its scalar proof uses a false \(\rho\)-bound and an unproved monotonicity assertion.

So the proof is **not complete yet**. The correct final target is narrower and precise: prove the large-order endpoint Bessel-envelope lemma \((L)\). Once \((L)\) is established, the direct Sonin localization immediately completes the full KKT conjecture for all real \(\alpha,\beta\ge0\), without integer-parameter representation theory.
